# DRONE CONSOLE FRONTEND
ระบบ Drone Monitoring & Logging Console เพื่อแสดงข้อมูล Drone และบันทึกอุณหภูมิ พร้อมแสดง Logs แบบ Realtime

## Project จะมี 3 ส่วนหลัก
1. View ConFig 
จะแสดงข้อมูลในปัจจุบันของโดรนที่เชื่อมต่ออยู่ 
ข้อมูลที่แสดงจะมี : Drone ID, Drone Name, Light, Country

2. LogTemperature 
จะมีไว้ให้ user ใส่ค่าอุณหภูมิ
จะใช้งานโดยการกรอกค่าอุณภูมิลงไป แล้วหลักจากนั้นกด Sunmit

3. ViewLogs
จะแสดงอุณหภูมิที่ user กรอกในหน้า LogTemperature 
ข้อมูลจะแสดงเป็นตารางเกี่ยวกับรายละเอียดทั้งหมดที่บันทึกไว้
